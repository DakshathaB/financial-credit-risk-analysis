# Financial Credit Risk Analysis

## Project Overview
This project focuses on analyzing customer financial data to identify credit risk levels and support loan decision-making.

## Tools Used
- Excel
- SQL
- Power BI

## Steps Performed
- Data cleaning and preprocessing
- Risk segmentation (high, medium, low)
- Data analysis using SQL
- Dashboard creation in Power BI

## Result
- Identified risk distribution among customers
- Provided insights for better loan approval decisions

## Conclusion
This project helps financial institutions reduce risk and improve decision-making using data analysis.
